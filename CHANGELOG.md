# Changelog

All notable changes to the SafeLabs project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-12

### Added
- Initial MVP release
- ESP32 firmware with DHT22, MQ-135, PIR sensor simulation
- Firebase Realtime Database integration
- Streamlit dashboard with real-time monitoring
- Google Gemini AI-powered insights
- AC/cooling system remote control
- Historical data analytics with Plotly charts
- Node.js Express backend with REST API
- Wokwi simulation support
- Comprehensive documentation in docs/ folder
- Security guidelines and credential templates
- Environment variable configuration

### Security
- Gitignore for sensitive credentials
- Config template files (.env.example, config.h.example)
- Firebase service account protection
